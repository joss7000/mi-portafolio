Mis Proyectos

01

Investigación: Estructura y Jerarquía Visual en el Diseño

Investigación académica sobre principios fundamentales de diseño web, incluyendo agrupación de elementos, jerarquía visual, espaciado en blanco, presentación del texto, técnicas de categorización en tarjetas (Card Sorting), prototipado y wireframes.

Diseño UX/UI Wireframes Card Sorting Prototipado Investigación
02

Sitio Web UNACHI - Universidad de Chiriquí

Desarrollo de página web informativa para la Universidad Autónoma de Chiriquí (UNACHI) con diseño desde cero en PHP. Incluye menú principal de navegación, información sobre misión, visión, servicios, programas académicos y planes de estudio.

HTML CSS PHP Diseño Web
03

Portal Web Municipio de Alanje

Sitio web oficial para el Municipio de Alanje desarrollado con HTML/CSS/ASP.NET. Presenta información institucional completa incluyendo inicio, visión y misión, servicios prestados, historia, departamentos municipales, contáctenos y ubicación geográfica.

HTML CSS ASP.NET Servidor Apache
04

Junta Comunal Las Lomas + Taller Kali Linux

Portal web para la Junta Comunal de Las Lomas con secciones de transparencia y servicios ciudadanos. Incluye taller práctico de ciberseguridad usando Kali Linux en VirtualBox para realizar pruebas de penetración (ataques DDoS, escaneo NMAP, análisis con Wireshark) en red LAN simulada entre servidores de compañeros.

HTML/ASP.NET Kali Linux VirtualBox NMAP Wireshark DDoS Ciberseguridad
05

Parcial 1: Centro de Lenguas UTP

Sitio web completo para el Centro Especializado en Lenguas (CEL) de la Universidad Tecnológica de Panamá. Diseño desde cero con HTML/CSS/ASP, incluyendo menú de navegación interna, cursos ofrecidos, ubicación, horarios, servicios, misión y visión. Réplica funcional del sitio oficial institucional.

HTML CSS ASP JavaScript
06

Succesos y Más - Organización de Eventos

Plataforma web profesional para empresa organizadora de eventos. Incluye página de inicio con noticias y presentación corporativa, catálogo de productos con precios, sección de servicios para diferentes tipos de eventos, galería de clientes y formulario de contacto.

HTML CSS JavaScript Diseño Responsivo
07

Investigación: XAMPP y MySQL con Prototipo RHODE

Investigación técnica sobre Base de Datos (MySQL) con PHP, incluyendo conceptos, requerimientos, configuración, archivos/entornos, puertos y replicación en otros SGBD. Desarrollo práctico de un prototipo web de gestión con XAMPP, integración con phpMyAdmin y demostración en vivo con capturas de implementación.

XAMPP MySQL PHP phpMyAdmin SGBD Investigación
08

Chinos Café - Sistema POS y Gestión

Sistema web completo de punto de venta (POS) para cafetería desarrollado en PHP con base de datos MySQL. Funcionalidades: gestión CRUD de productos, control de inventario y stock, procesamiento de ventas, generación de facturas, sistema de proveedores y administración de inventario con servidor MySQL y archivos.

PHP MySQL CRUD POS Facturación
09

NIBARRA - Sistema de Mantenimiento (Parcial 2)

Sistema web empresarial completo para gestión de mantenimientos de PYME. Incluye: Header/Footer con estructura profesional, CRUD de equipos con base de datos local replicada a servidor externo (Supabase), calendario mensual con agenda de mantenimientos, tablero Kanban (Por Hacer, En Espera, En Revisión, Terminada) con tipos de mantenimiento (predictivo, preventivo, correctivo) y barra de progreso visual. Sistema de acceso controlado, ChatBot integrado, protección de código y simulación de red LAN/VPN con 2 routers conectando Chiriquí-David con servidor remoto.

PHP MySQL Supabase CRUD Kanban ChatBot Calendario Red VPN Autenticación
